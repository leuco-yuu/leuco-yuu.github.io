# 表格自动适配

本博文对表格进行了部分优化。

- **宽表格**：超出容器宽度时自动添加水平滚动条
- **窄表格**：自动拉伸填满容器宽度
- **焦点样式**：滚动区域可聚焦，有 focus outline
- **强制滚动/拉伸**：本博客中，对于宽度比较大的表有横向滚动策略与竖向拉伸策略，项目自动进行选择。但可以通过以下方式强制设置：
  - **单表设置**：该方式优先级最高，在每个表格的后面新增以下条目以实现该效果：
    - `{tableMode="scroll"}`：强制为横向滚轮。
    - `{tableMode="stretch"}`：强制为竖向拉伸。

  - **全文设置**：该方式优先级次高，在正文头数据中加入`tableMode: "scroll"`或`tableMode: "scroll"`以实现对全文表格的强制。

**格式案例 1**：

```markdown
| 攻击类型 | 目标协议/服务 | 典型防御手段 |
| :--- | :---: | :--- |
| SYN Flood | TCP | SYN Cookie、增大积压队列 |
| DNS 欺骗 | UDP 53 | DNSSEC、随机源端口 |
| ARP 欺骗 | ARP | 静态ARP表、DAI（动态ARP检测） |
```
