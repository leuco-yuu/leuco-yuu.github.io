# 批注与提示

## 标准Alert提示框


基于 Hugo 标准 **GFM Alert** 语法 `> [!TYPE]`，本博客做了完整的视觉定制，5 种不同颜色：

| 类型 | 写法 | 颜色 | 默认标题 |
|:---|:---|:---|:---|
| **NOTE** | `> [!NOTE]` | 蓝色调 | "Note" |
| **TIP** | `> [!TIP]` | 绿色调 | "Tip" |
| **IMPORTANT** | `> [!IMPORTANT]` | 紫色调 | "Important" |
| **WARNING** | `> [!WARNING]` | 橙色调 | "Warning" |
| **CAUTION** | `> [!CAUTION]` | 红色调 | "Caution" |

- **可折叠 Alert**：在 `[!TYPE]` 后加 `+`（默认展开）或 `-`（默认折叠）；
- **交互效果**：hover 上浮动效、阴影增强，可折叠的带旋转箭头。


### NOTE

---

**格式案例**：

```markdown
>[!NOTE]
>This is a case demonstrating the NOTE format style.
```

**渲染样式**：

> [!NOTE] 
>
> This is a case demonstrating the NOTE format style.

## 本博客项目新增批注格式

### ANNOT


这是本博客新增的 GFM Alert 扩展类型，语法为 `> [!ANNOT]`：

- **图标**：左侧显示自定义的批注图标；
- **自动编号**：同一篇文章内的批注块会自动从 1 开始编号；
- **设计**：左侧有竖线分隔，底色为主色调半透明，视觉上与正文区隔明显。


---

**格式案例**：

```markdown
>[!ANNOT]
>This is a case demonstrating the ANNOT(FIRST) format style.

>[!ANNOT]
>This is a case demonstrating the ANNOT(SECOND) format style.
```

**渲染样式**：

> [!ANNOT] 
>
> This is a case demonstrating the ANNOT(FIRST) format style.

>[!ANNOT]
>This is a case demonstrating the ANNOT(SECOND) format style.

---
