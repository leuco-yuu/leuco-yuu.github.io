# 引用外部论文和标准时的映射规范

1. 首次引用外部方案时，建立独立的“原文符号映射表”。
2. 映射表至少包含：维数、模数、样本数、模块秩、秘密、误差、公钥、私钥、密文、签名、挑战、响应、随机币和失败事件。
3. 原论文若使用 $A\in\mathbb Z_q^{m\times n}$ 与 $\mathbf b=A\mathbf s+\mathbf e$，正文应统一转写为本表的列样本方向，或整章明确采用原方向，二者不可混用。
4. 标准文档中的字段名、字节序列名和算法名应保持标准拼写；数学推导可映射到本表记号。
5. 代码变量应在实现章节提供“代码变量 → 数学对象”映射，例如 `rho -> \mathsf{seed}_A`、`sigma -> \mathsf{seed}_e`，不得因代码沿用 `sigma` 就将其误解为 Gaussian 标准差。
6. 若同一方案的论文版、提交版和标准版参数不同，应加版本上标或参数集标识，例如 $q^{\rm paper}$、$q^{\rm std}$，或直接使用 $\mathsf{paramID}$。
