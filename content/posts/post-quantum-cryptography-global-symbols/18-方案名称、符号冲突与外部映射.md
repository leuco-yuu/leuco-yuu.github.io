# 方案名称、符号冲突与外部映射

## 方案与算法名称

1. 原语类别使用正体大写缩写：$\mathsf{PKE}$、$\mathsf{KEM}$、$\mathsf{SIG}$、$\mathsf{ZK}$。
2. 具体方案名称保留官方拼写，例如 Kyber、Dilithium、Falcon。
3. 算法接口统一使用 $\mathsf{KeyGen}$、$\mathsf{Enc}$、$\mathsf{Dec}$、$\mathsf{Encaps}$、$\mathsf{Decaps}$、$\mathsf{Sign}$、$\mathsf{Vrfy}$。
4. 方案整体使用 $\Pi$ 或 $\Sigma$，不得用算法接口名称表示完整方案。

## 必须避免的冲突

|冲突符号|统一处理|
|-|-|
|$m$|全局表示样本数；消息使用 $\mu$|
|$s$ 与 $\sigma$|$s$ 表示格 Gaussian 参数，$\sigma$ 表示标准差|
|$K$ 与 $\mathbb{K}$|$K$ 表示密钥，$\mathbb{K}$ 表示数域|
|$\mathbf{G}$ 与 $\mathbf{G}_{\rm code}$|分别表示 Gadget 矩阵与编码生成矩阵|
|$\mathsf{H}$ 与 $\mathbf{H}$|分别表示哈希函数与矩阵|
|$\sigma$ 与签名|签名对象优先写 $\mathsf{sig}$|
|$\lambda_i$|格逐次极小值；矩阵特征值应另加语义标记|

## 外部文献映射

引用外部论文时，在首次出现处给出映射，例如：

|原文记号|本表记号|
|-|-|
|$A\in\mathbb{Z}_q^{m\times n}$|$\mathbf{A}^{\top}\in\mathbb{Z}_q^{m\times n}$|
|$\sigma$（离散 Gaussian 参数）|$s$|
|$m$（消息）|$\mu$|

保留原文推导时不得在同一公式链中混用原记号与本表记号。
