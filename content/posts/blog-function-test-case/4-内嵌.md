# Inframe 嵌入页
本部分功能为项目独有，Hugo 没有内置 iframe 支持。博客通过自定义 shortcode inframe.html 实现：

- **参数驱动**：src（必填）、title（默认=src）、ratio（默认 16:9，支持 16:9 和 16/9 两种写法）
- **比例方案**：不用 aspect-ratio CSS 属性（兼容性不稳），改用经典的 padding-bottom 百分比法——Hugo 模板层用 div + mul 实时计算 height/width × 100，生成 inline style。ratio="3:1" 就是 33.33%，无需预设比例。
- **卡片 UI**：仿 macOS 窗口——三色圆点（首个用 --color-primary）、monospace 标题、复制 URL + 新窗口打开按钮，边框使用主题色；
- **复制交互**：事件委托 + navigator.clipboard（execCommand 降级），1.5s toast 反馈。

```demo {title="Inframe 内嵌网页展示" render="md"}
{{</* inframe src="https://leuco-yuu.github.io" title="Leuco's Monologue" ratio="16:9" */>}}
```
