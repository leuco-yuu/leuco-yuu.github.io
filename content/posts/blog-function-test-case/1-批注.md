# 警示框与批注

基于 Hugo 标准 **GFM Alert** 语法 `> [!TYPE]`，本博客做了完整的视觉定制，5 种不同颜色+1种动态主题色：

| 类型              | 写法             | 颜色   | 默认标题 |
| :---------------- | :--------------- | :----- | :------- |
| **NOTE**          | `> [!NOTE]`      | 蓝色调 | “注意”   |
| **TIP**           | `> [!TIP]`       | 绿色调 | “提示”   |
| **IMPORTANT**     | `> [!IMPORTANT]` | 紫色调 | “重要”   |
| **WARNING**       | `> [!WARNING]`   | 橙色调 | “警告”   |
| **CAUTION**       | `> [!CAUTION]`   | 红色调 | “小心”   |
| **ANNOT**（新增） | `> [!ACCNOT]`    | 主题色 | “批注”   |

- **可折叠 Alert**：在 `[!TYPE]` 后加 `+`（默认展开）或 `-`（默认折叠）；
- **交互效果**：hover 上浮动效、阴影增强，可折叠的带旋转箭头；
- **同类编号**：相同类型 Alert 数量 ≥2 条时自动独立编号；
- **Alert 跳转**：任意类型 Alert 数量 ≥2 条时 Alert 框出现导航按钮以快速跳转。
