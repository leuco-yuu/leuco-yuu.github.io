# 公式渲染

本博客项目已集成 KaTeX，支持行内和块级数学公式：

- **行内公式**：`$...$` 或 `\(...\)`；
- **块级公式**：`$$...$$` 或 `\[...\]`；
- **复制支持**：通过集成了 `copy-tex` 插件，点击公式可复制完整 LaTeX 源码；
- **自动检测**：页面有数学内容时自动加载 KaTeX 资源。

---

**格式案例**：

```markdown
$$
\begin{aligned}
& QS([29,10,14,37,13,25,1,5]) = QS([5,1]) \oplus [10] \oplus QS([14,13,25]) \oplus [29] \oplus [37] \\
& \text{where, } QS([5,1]) = [1] \oplus [5] \oplus [] = [1,5] \\
& \phantom{\text{where, }} QS([14,13,25]) = [13] \oplus [14] \oplus [25] = [13,14,25]
\end{aligned}
$$
```
**渲染样式**：
$$
\begin{aligned}
& QS([29,10,14,37,13,25,1,5]) = QS([5,1]) \oplus [10] \oplus QS([14,13,25]) \oplus [29] \oplus [37] \\
& \text{where, } QS([5,1]) = [1] \oplus [5] \oplus [] = [1,5] \\
& \phantom{\text{where, }} QS([14,13,25]) = [13] \oplus [14] \oplus [25] = [13,14,25]
\end{aligned}
$$
