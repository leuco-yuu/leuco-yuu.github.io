# 图片与特色画廊

每张图片都带有交互工具层：

- **Zoom** — 右上角放大按钮，点击进入灯箱模式全屏查看；
- **Download** — 右上角下载按钮；

- **外部/本地自适应**：本地图片走 Hugo resources.Get 管道自动转 WebP；外部图片原样输出
- **操作按钮**：每张图 hover 显示缩放 + 下载悬浮按钮
- **Lightbox + Gallery**：lightbox.enabled 控制点击放大，gallery.enabled 控制多图 justified 布局，全部通过 smart-gallery.min.js 驱动
