## Project Overview

这个项目是当前博客本身：使用 Hugo 生成静态页面，主题保持原样，所有个性化内容都放在站点配置、内容和静态资源中。

## Features

- 文章、项目、分类、标签、系列和归档页面
- GitHub Pages Actions 自动构建
- 首页作者卡片和精选项目
- KaTeX、Mermaid、代码高亮和搜索支持

## Technologies Used

- Hugo Extended
- Hugo Narrow
- GitHub Pages
- Markdown
- GitHub Actions

## Notes

后续重点是补齐真实文章和项目，并把分类、标签、系列维持在一个可长期扩展的结构里。
