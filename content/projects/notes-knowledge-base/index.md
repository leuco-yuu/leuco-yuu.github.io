## Project Overview

Notes Knowledge Base 关注的是长期笔记的组织方式。它把临时记录逐步整理成文章、清单、系列和项目复盘，避免知识只停留在零散文件里。

## Features

- 用统一 front matter 记录主题、标签和系列
- 将实验记录沉淀为可发布文章
- 为项目复盘建立固定模板
- 用 Hugo 搜索和归档能力提升可发现性

## Technologies Used

- Markdown
- Hugo
- Git
- GitHub Pages

## Notes

这个项目会优先解决“如何命名、如何分类、何时发布”的问题。工具只是其次，稳定的整理规则更重要。
